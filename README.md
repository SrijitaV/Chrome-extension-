## 📖 Overview

LeetPilot is an AI-powered Chrome Extension that enhances the LeetCode experience by bringing learning, organization, and revision tools directly into your coding workflow.

Instead of switching between multiple tabs for notes, AI explanations, bookmarks, and revision tracking, LeetPilot keeps everything in one place through a clean Side Panel interface.

The goal is to help developers understand problems better, build long-term knowledge, and prepare more effectively for coding interviews.
