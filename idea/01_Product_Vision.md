# CodePilot AI

## Vision

CodePilot AI is an AI-powered Chrome Extension designed to help software engineers and students solve coding problems more effectively.

Instead of switching between multiple tabs for notes, hints, explanations, and revision, CodePilot AI brings everything directly into the coding platform.

The extension focuses on improving learning rather than giving direct solutions.

---

## Goals

- Learn coding patterns faster
- Organize DSA preparation
- Generate intelligent hints
- Track revision progress
- Build a personal knowledge base
- Reduce context switching

---

## Target Users

- Students preparing for placements
- Software Engineers
- Competitive Programmers
- Interview candidates
