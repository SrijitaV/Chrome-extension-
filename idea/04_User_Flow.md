# CodePilot AI - User Personas

## Overview

CodePilot AI is designed for developers preparing for coding interviews and improving problem-solving skills.

The product focuses on helping users learn, organize, and revise coding problems rather than providing direct solutions.

## Persona 1 - Placement Student

### Profile

- College Student
- Preparing for internships and placements
- Solves DSA problems daily
- Uses LeetCode frequently

### Goals

- Crack coding interviews
- Learn patterns
- Revise efficiently
- Keep notes organized

### Pain Points

- Forgets solutions after a few weeks
- Revisits the same mistakes
- Keeps notes in multiple places
- Gets stuck on problems for long periods
- Switches between many browser tabs for explanations

### How CodePilot AI Helps

- Saves notes for each problem
- Provides AI-generated hints
- Tracks revision progress
- Organizes problems using tags

## Persona 2 - Competitive Programmer

### Profile

- Practices coding contests regularly
- Solves many problems every week

### Goals

- Learn new algorithms
- Organize solved problems
- Build a personal reference library

### Pain Points

- Difficult to remember previously solved problems
- No structured revision process

### How CodePilot AI Helps

- Searchable notes
- Bookmarks
- Pattern classification
- Revision reminders

## Persona 3 - Software Engineer

### Profile

- Working professional
- Preparing for job switch

### Goals

- Refresh DSA concepts
- Practice interview questions
- Review important patterns

### Pain Points

- Limited study time
- Needs quick explanations
- Wants efficient revision

### How CodePilot AI Helps

- AI explanations
- Flashcards
- Dashboard
- Saved notes

# Primary Use Cases

- Solving a new coding problem
- Revising previously solved problems
- Taking personal notes
- Getting AI hints
- Understanding algorithm patterns
- Searching past notes
- Preparing before interviews

# Out of Scope (Version 1)

The following features are intentionally excluded from the first release:

- Solving problems automatically
- AI code generation
- Competitive programming submissions
- Code execution
- User authentication
- Cloud synchronization
- Team collaboration
