# CodePilot AI Mind Map

The project consists of five primary modules.

## 1. Learning

Purpose

Help users understand coding problems instead of exposing direct solutions.

Features

- AI Hint
- Complexity Analysis
- Pattern Detection
- Edge Case Suggestions

---

## 2. Organization

Purpose

Maintain a personal DSA knowledge base.

Features

- Notes
- Tags
- Bookmarks
- Search

---

## 3. AI Assistant

Purpose

Use LLMs to accelerate learning.

Features

- Explain
- Summarize
- Flashcards
- Revision Suggestions

---

## 4. Smart Features

Purpose

Provide automation.

Features

- Context Awareness
- Smart Recommendations
- Privacy First

---
